

## Getting Started

'''
pip3 install pipenv
pipenv install
'''

Activate the pipenv:

'''
pipenv shell
'''

or use

'''
pipenv run
'''

to run commands inside the project directory.

### Running the tests

To run the package tests, change into the pipenv and run pytest:

'''
pytest
'''

Alternatively, you can use the setup.py script:

'''
python3 setup.py test
'''
